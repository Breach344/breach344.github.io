FGO Duel(Video Game)

If you don't know the rules they're on the official site. It's still a work in progress but I wanted to show off what I'm working on. When you make an action it checks many variables before deciding what to do. Your pieces on the board are assigned individual classes that get removed and added from the tiles when you move in order to keep track of their locations. The turn phases are done with a number counter that changes based on your actions. Only certain actions are allowed for each phase(only when the turn counter variable is at certain values). Each card has an individual id to identify its type, power, and game piece. If all cards used in battle have a type in common they create a chain with varying abilities that could last several turns which I did by using variables that tick down counters when turns end. When a unit attempts to land on a space with an enemy class on it then it will start the battle function where it checks for card power, card chains, and the counter variables to see who wins. All the functionality is worked out and tested. There is just data entry, optimizing/organizing the code, and styling to finish.
